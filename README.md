# 🗑️ TrashCompactor iOSアプリ

卒業制作

iOSアプリとArduinoをBLE（HM-10）で連携させ、ごみ箱圧縮機のモーターを遠隔操作できるシステム

iPhoneアプリから“圧縮開始”コマンドを送信し、Arduino上のBLDCモーターが約30秒間の圧縮サイクルを実行

安全性確保のため、圧縮動作中の重複コマンド防止やステータス表示などの制御機能も実装

---

## 🔧 主な機能

- 🔵 HM-10 BLEモジュールの自動スキャンおよび接続
- 📡 BLEコマンド送信による圧縮動作の制御
- ⏱ 圧縮動作中の状態管理および重複実行の防止

---

## 🧠 プロジェクト構成

### BLEMotorManager

- CoreBluetoothを用いたBLE通信マネージャ
- HM-10デバイスの検索および接続管理
- サービス（FFE0）およびキャラクタリスティック（FFE1）の探索
- 圧縮開始コマンド `"C"` の送信
- Test Mode提供（実際のBLE通信なしでロジックテストが可能）

### ViewController

- ユーザーインターフェース（UI）の構成およびボタンイベント処理
- BLE接続状態および圧縮進行状態の表示
- 圧縮動作中の重複コマンド送信防止
- Arduinoの動作時間を基準にしたタイマーによる圧縮完了処理

---

## ⚙️ 使用技術

- SnapKit
- CoreBluetooth
- Bluetooth Low Energy（BLE）
- Arduino（外部ハードウェア連携）
